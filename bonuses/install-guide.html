מתקינים Claude Desktop
בחמישה צעדים

מההורדה ועד השיחה הראשונה. בלי ידע טכני — רק עוקבים אחרי הצעדים.

1

הורדת האפליקציה

היכנסו לאתר הרשמי והורידו את הגרסה למחשב שלכם (Mac או Windows).

  • גלשו ל-claude.ai/download
  • האתר מזהה את סוג המחשב לבד ומציע את הקובץ הנכון
  • לחצו Download ושמרו את הקובץ
צילום מסך — capture-needed עמוד ההורדה עם כפתור Download מודגש
2

התקנה במחשב

פותחים את הקובץ שהורד ומתקינים — בדיוק כמו כל אפליקציה.

  • Mac: פתחו את קובץ ה-.dmg וגררו את Claude לתיקיית Applications
  • Windows: הריצו את קובץ ה-.exe ואשרו את ההתקנה
  • פתחו את האפליקציה מתפריט ההתחלה / Launchpad
צילום מסך — capture-needed חלון ההתקנה (גרירה ל-Applications)
3

התחברות / יצירת חשבון

בפעם הראשונה תתבקשו להתחבר. אפשר עם חשבון Google או עם אימייל.

  • לחצו Continue with Google או הזינו אימייל
  • אשרו את הקוד שנשלח אליכם למייל
  • זהו — אתם בפנים
לא צריך כרטיס אשראי כדי להתחבר. אפשר להתחיל בחינם ולשדרג בהמשך.
צילום מסך — capture-needed מסך ההתחברות (Continue with Google / אימייל)
4

רכישת מנוי (מומלץ)

המנוי בתשלום (Pro) פותח שימוש נרחב יותר, קבצים גדולים, ויכולות מתקדמות — שווה לבעל עסק שעובד עם זה ביומיום.

  • בתוך האפליקציה לחצו על שמכם → Settings → Billing / Plan
  • בחרו Upgrade to Pro
  • הזינו פרטי תשלום ואשרו — המנוי נכנס לתוקף מיד
צילום מסך — capture-needed מסך השדרוג ל-Pro עם המחיר
5

השיחה הראשונה

פתחו שיחה חדשה ונסו את אחד הפרומפטים מחבילת התלמיד. למשל:

  • "תכין לי טבלת מעקב לקוחות פשוטה"
  • "סכם לי את דף הבנק המצורף"
  • "כתוב לי פוסט לינקדאין על [נושא]"
