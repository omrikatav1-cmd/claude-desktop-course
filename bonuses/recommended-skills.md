# Skills מומלצים להכיר

מעבר ל-30 ה-skills בחבילה, ל-Claude יש יכולות מובנות שכדאי לבעל עסק להכיר. הנה רשימה מסודרת — מה זה, ומתי להשתמש.

## מסמכים — היכולות הכי שימושיות ביומיום

- **Excel / Sheets (xlsx)** — בניית טבלאות וגיליונות. תמיד בקשו `rightToLeft` לעברית. מושלם ל: סיכומי הוצאות, מעקב לקוחות, תמחור.
- **Word (docx)** — מסמכים מעוצבים: הצעות מחיר, חוזים, מכתבים רשמיים.
- **PowerPoint (pptx)** — מצגות מהירות ללקוח / למשקיע מתוך נקודות שתתנו.
- **PDF** — קריאה, חילוץ טקסט וטבלאות, מיזוג ופיצול קבצים, מילוי טפסים. מצוין לעבודה מול חשבוניות וחוזים סרוקים.

> טיפ: כשאתם מצרפים PDF סרוק (תמונה), בקשו במפורש "תקרא עם OCR בעברית".

## תוכן ושיווק

- **כתיבת תוכן בעברית** — Claude כותב פוסטים, מיילים ועמודי אתר בעברית טבעית. תנו לו טון וקהל.
- **copywriting / הצעות ערך** — חידוד מסרים שיווקיים, כותרות, CTA.
- **social-content** — תכנון ותזמון תוכן לרשתות, התאמת אותו רעיון לכמה פלטפורמות.

## אתרים וכלים (גם בלי לדעת קוד)

- **בניית דף נחיתה / אתר תדמית** — תארו עסק, וקבלו אתר RTL מעוצב בקובץ אחד.
- **web artifacts / כלים אינטראקטיביים** — מחשבונים, טפסים, טבלאות מעקב — דברים שעובדים בדפדפן.
- **frontend-design** — כשרוצים שהתוצאה תיראה יוקרתי ולא "AI גנרי".

## נגישות ותקינה (חשוב בישראל)

- **israeli-accessibility-compliance** — התאמת אתר לתקן הישראלי IS 5568 ו-WCAG. חובה חוקית לעסקים עם אתר.
- **hebrew-rtl-best-practices** — לוודא שהעברית מסתדרת נכון מימין לשמאל.

## חיבורים (Connectors / MCP) — מתקדם

Claude Desktop יכול להתחבר לכלים שאתם כבר משתמשים בהם, כדי לעבוד ישירות מולם:

- **Google Drive / Gmail / Calendar** — לקרוא קבצים, לנסח מיילים, לתאם פגישות.
- **Notion** — לנהל מסמכים ומשימות.
- **Shopify** — לנהל חנות, מוצרים והזמנות.

> חיבורים מוגדרים ב-**Settings → Connectors**. התחילו מאחד שאתם באמת משתמשים בו.

## איך מגלים skills נוספים

בתוך Claude פשוט תשאלו: **"יש לך skill בשביל [משימה]?"** או **"איך אני עושה [משימה]?"**. אם קיים skill מתאים — Claude יציע אותו. אפשר גם לבקש: "הצג לי אילו skills זמינים לי".
