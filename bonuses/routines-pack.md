# Routines Pack — 3 אוטומציות שכל אחד יוצא איתן מהסדנה

> **חלק מחבילת הבונוסים של קורס Claude Desktop.**
> Routine = משימה ש-Claude מריץ **לבד, לפי לו"ז** (cron) — מקומית או בענן. אתה מגדיר פעם אחת, והיא רצה בלעדיך.

המטרה: לצאת מהסדנה עם **3 רוטינות עובדות**. הראשונה (WhatsApp/Green-API) היא הדגל — היא מראה את הכוח של "Claude עם ידיים" שמדבר עם העולם האמיתי.

---

## רוטינה 1 — הודעת WhatsApp אוטומטית דרך GREEN-API ⭐

**מה זה עושה:** שולח הודעת WhatsApp לפי לו"ז — תזכורת, סיכום יומי, או פולואפ ללקוח — בלי שתיגע בטלפון.

### ⚠️ לפני הכל — אזהרת ToS (חובה לקרוא)
GREEN-API הוא **gateway לא-רשמי**: הוא מחבר חשבון WhatsApp רגיל דרך אוטומציה. הוא **מפר את תנאי השימוש של WhatsApp**, ו-Meta יכולה לחסום את המספר **לצמיתות, בלי התראה**. לכן:
- **מתאים ל:** אוטומציה אישית, דמו, הוכחת היתכנות — על מספר שלא אכפת לך ממנו.
- **לא מתאים ל:** production מול לקוחות משלמים. לזה יש את המסלול הרשמי (Meta Cloud API / 360dialog).
- השתמש ב**מספר ייעודי**, לא הנייד הראשי שלך. מספר שעובר ל-API יוצא מאפליקציית WhatsApp הרגילה.

### הקמה (פעם אחת)
1. פתח חשבון ב-GREEN-API והקם **instance** חדש.
2. סרוק את ה-QR מהמספר הייעודי דרך WhatsApp — זה מקשר את המספר ל-instance.
3. העתק את `idInstance` ו-`apiTokenInstance`.
4. שמור אותם כ-**env vars server-side** — `GREENAPI_ID_INSTANCE`, `GREENAPI_API_TOKEN`. **לא בקוד, לא בצ'אט.**
5. שלח הודעת בדיקה ואשר שהיא מגיעה.

### הקריאה בפועל (מה שקורה מתחת)
```
POST https://api.green-api.com/waInstance{idInstance}/sendMessage/{apiToken}
Content-Type: application/json

{ "chatId": "972501234567@c.us", "message": "ההודעה שלך" }
```

### הפרומפט ל-Routine (מדביקים ב-Claude)
```
צור routine שרץ כל יום ב-09:00.
המשימה: לשלוח הודעת WhatsApp דרך GREEN-API.
- קרא את GREENAPI_ID_INSTANCE ו-GREENAPI_API_TOKEN מ-env (אל תבקש ממני להדביק אותם).
- שלח POST ל-https://api.green-api.com/waInstance{id}/sendMessage/{token}
  עם body: { chatId, message }.
- chatId היעד: [המספר שלך]@c.us
- תוכן ההודעה: סיכום המשימות שלי להיום מתוך [המקור — קובץ/קונקטור].
לפני שאתה בונה — הצג לי Plan ובקש אישור.
```

## רוטינה 2 — בריפינג בוקר יומי

**מה זה עושה:** כל בוקר Claude מרכיב לך סיכום קצר — מה ביומן היום, מה המשימות הפתוחות, ולידים חדשים — ושם אותו בקובץ/שולח אליך.

### הפרומפט
```
צור routine יומי ב-07:30.
אסוף: אירועי היומן של היום (קונקטור Calendar), משימות פתוחות מ-[המקור],
ולידים חדשים מ-[המקור]. סכם ל-5 שורות עם 3 העדיפויות של היום.
שמור ל-briefings/YYYY-MM-DD.md. אם מחובר WhatsApp — שלח לי גם שם.
```
> אפשר לחבר את רוטינה 1 לרוטינה 2: הבריפינג נשלח אליך ב-WhatsApp.

## רוטינה 3 — דוח עסקי שבועי

**מה זה עושה:** פעם בשבוע Claude מפיק סקירה עסקית (מדדים מול שבוע קודם ומול יעד) — מתחבר ל-skill `weekly-business-review` מהחבילה.

### הפרומפט
```
צור routine שבועי, ראשון ב-08:00.
הפעל את הזרימה של weekly-business-review על גיליון המדדים ב-[נתיב/קונקטור].
הפק טבלת WBR + 3-5 פעולות לשבוע. שמור ל-reports/ ושלח אליי סיכום.
```

## איפה מריצים — מקומי מול ענן
- **מקומי (Local):** רץ על המחשב שלך. חינמי, אבל רק כשהמחשב דולק.
- **ענן (Remote/Cloud):** רץ על שרת של Claude — גם כשהמחשב כבוי. אידיאלי לרוטינות בוקר/לילה.

## כלל זהב
כל רוטינה שנוגעת בעולם האמיתי (שולחת הודעות, מוציאה כסף, מוחקת) — **תבדוק אותה ידנית פעם אחת לפני שמשחררים על אוטומט**, ותתחיל ממקור/מספר שלא קריטי.
