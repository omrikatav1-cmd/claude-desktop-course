מאת **טל מובשוביץ ✕ עומרי כתב**.

> איך לחסוך 20–40 שעות בחודש בעזרת סוכני בינה מלאכותית.

## איך פותחים

- **אונליין:** דרך GitHub Pages — הלינק נטען אוטומטית בדפדפן.
- **מקומית:** מורידים את ה-repo ופותחים את `index.html`. למצגת, עדיף להריץ שרת מקומי:
  ```bash
  python3 -m http.server 8000
  # ואז: http://localhost:8000/presentation/index.html
  ```
