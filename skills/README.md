30 כישורים (skills) מוכנים לבעל עסק. כל skill הוא תיקייה אחת עם קובץ `SKILL.md` — נוהל קצר בעברית ש-Claude טוען אוטומטית כשהשיחה רלוונטית. אין מה להתקין חוץ מלהעתיק תיקייה.

החבילה מחולקת לשתי קבוצות:

- **`omri/`** — צמיחה, שיווק, תוכן ולקוחות (15).
- **`tal/`** — כספים, אדמיניסטרציה, תפעול ומסמכים (15).

## איך מתקינים skill ב-Claude Desktop

1. **הורידו** את התיקייה של ה-skill (התיקייה שמכילה את `SKILL.md`).
2. **מצאו את תיקיית ה-skills** של Claude Desktop:
   - בתוך האפליקציה: **Settings → Capabilities / Skills** → שם מופיע הנתיב לתיקיית ה-skills במחשב, וכפתור לפתיחתה.
3. **העתיקו** את תיקיית ה-skill אל תוך תיקיית ה-skills (כל skill בתיקייה נפרדת, עם `SKILL.md` בפנים).
4. **הפעילו מחדש** את Claude Desktop אם צריך.
5. **זהו** — Claude מזהה את ה-skill לבד ומשתמש בו אוטומטית כשהשיחה רלוונטית. אפשר גם פשוט לבקש בשם, למשל: "תכין לי FAQ" או "סכם את הפגישה".

> טיפ: אפשר להעתיק את **כל** התיקייה (`omri/` או `tal/`) בבת אחת — כל ה-skills שבתוכה ייטענו יחד.

### מה זה בעצם skill?

קובץ טקסט קצר שמלמד את Claude **נוהל עבודה** — מה לאסוף, איך לבנות את הפלט, ומה הכללים (למשל "אל תמציא נתון"). זה לא קוד ולא תוכנה — רק הוראות. אתם תמיד מקבלים **טיוטה לאישור**, אף פעם לא פעולה אוטומטית על כסף או על לקוחות.
