# הודעת עדכון ללקוח — הנוהל

מנסח הודעות עדכון/מעקב קצרות ומקצועיות ללקוח, כדי שהוא תמיד ירגיש מעודכן. הכל **טיוטה לאישור שלך**.

## שלב 1 — קלט
קבל את ההקשר: מי הלקוח, מה השלב/האירוע (יצאנו לדרך, אבן דרך הושלמה, מחכים לחומר ממנו), ומה הצעד הבא.

## שלב 2 — שלושה טונים
הפק את ההודעה ב-3 גרסאות לבחירה:
1. **רשמי** — ללקוח עסקי/חדש.
2. **חברי-מקצועי** — ללקוח מוכר.
3. **קצר** — לוואטסאפ.

כל גרסה כוללת: מה קרה → מה הצעד הבא → מה נדרש מהלקוח (אם בכלל).

## שלב 3 — Follow-up
אם זו תזכורת למי שלא הגיב — נסח אותה מנומסת, בלי לחץ, עם הצעה קונקרטית להמשך.

## שלב 4 — פלט
הצג בצ'אט את 3 הגרסאות (או את ה-follow-up).

## כללים
- אל תבטיח תאריך/תוצאה שלא נמסרו לך — `[להשלים]`.
- קצר ומכבד את זמן הלקוח.
- טיוטה לאישור שלך.
