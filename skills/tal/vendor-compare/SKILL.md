# השוואת ספקים — הנוהל

החלטת רכש טובה מתחילה בהשוואה הוגנת. הנוהל מסדר כמה הצעות לטבלה אחת שמקלה להחליט.

## שלב 1 — קלט
קבל את ההצעות (PDF/מייל/טקסט) ואת מה שחשוב לך (מחיר, זמן אספקה, אחריות, שירות).

## שלב 2 — טבלת השוואה
שורה לכל ספק, עמודות:
1. **מחיר** (ובסיס — חד-פעמי / חודשי).
2. **מה כלול** ומה לא.
3. **תנאים** — תשלום, ביטול, אחריות.
4. **יתרונות / חסרונות** עיקריים.

תא שלא מופיע בהצעה = `[חסר]`.

## שלב 3 — המלצה
מתחת לטבלה: המלצה מנומקת לפי מה שחשוב לך, כולל "שווה לבדוק עם הספק" אם חסר מידע.

## שלב 4 — פלט
1. שמור טבלה (xlsx עם rightToLeft / md).
2. בצ'אט: הצג את הטבלה וההמלצה.

## כללים
- אל תמציא מחיר/תנאי שלא בהצעה — `[חסר]`.
- ההמלצה נגזרת מהקריטריונים שלך, לא מהזול ביותר אוטומטית.
